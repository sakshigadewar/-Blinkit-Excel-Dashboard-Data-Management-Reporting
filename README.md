  ## 🎯 Objective  
The objective of this project is to design an interactive Excel dashboard that efficiently transforms raw retail data into meaningful insights. The dashboard aims to analyze sales performance, identify key trends, and provide a clear visualization of business metrics such as outlet performance, product categories, and customer preferences to support data-driven decision-making.

---
## 📂 Dataset  
The dataset used in this project contains retail data related to Blinkit operations, including sales, products, and outlet information.

### 📊 Data Includes:
- Item Details (Item Type, Fat Content, Item Identifier)  
- Sales Data (Total Sales, Average Sales)  
- Outlet Information (Outlet Size, Location, Establishment Year, Outlet Type)  
- Product Ratings  
- Number of Items  

### 🧹 Data Preparation:
- Cleaned missing and inconsistent values  
- Formatted data for analysis  
- Created calculated columns and measures for KPIs  

> Note: The dataset used is a sample dataset for learning and project purposes.

---
## 🚀 Key Features  
- 📊 Interactive Excel Dashboard for real-time data analysis  
- 📈 KPI Tracking: Total Sales, Average Sales, Number of Items, and Average Rating  
- 🏪 Outlet Performance Analysis by Size, Location (Tier 1, Tier 2, Tier 3), and Type  
- 🛒 Category-wise Sales Insights across different item types  
- 🥗 Fat Content Analysis (Low Fat vs Regular)  
- 📅 Trend Analysis based on Outlet Establishment Year  
- 🎛️ Dynamic Filters (Slicers) for customized data views  
- 📉 Visualizations using Pivot Charts, Bar Charts, and Donut Charts  
- ⚡ Automated calculations using Excel formulas and functions  
- 📌 User-friendly and visually appealing dashboard design

## 📊 Insights Derived  

- 🏪 Tier 3 outlets contribute the highest share of total sales, indicating strong demand in these locations  
- 🛒 Supermarket Type 1 outlets generate the maximum revenue among all outlet types  
- 🥗 Low Fat products account for a larger portion of total sales compared to Regular items  
- 🍎 Fruits & Vegetables and Snack Foods are the top-performing product categories  
- 📈 Sales show a steady trend based on outlet establishment year, highlighting business growth over time  
- 📍 Outlet size (Medium) contributes the highest sales compared to Small and High sizes  
- ⭐ Average ratings remain consistent across different outlet types, indicating stable customer satisfaction
